Design Notes

  • Each flag card shows: reason badge, student email, time ago, status
  • Expanded view shows full conversation context: student's question + flagged response + student comment
  • Status filter pills at top let admin focus on pending vs reviewed vs dismissed
  • "Pending" count badge in panel header draws attention
  • Reviewed cards are dimmed to reduce visual noise
  • Actions: "Mark Reviewed" (green) and "Dismiss" (gray)
