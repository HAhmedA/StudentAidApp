A. Hover-reveal icon (top-right)

A small flag icon (⚑) appears in the top-right corner of the message bubble when the user hovers over it. Clean by default, discoverable on interaction. This is the ChatGPT / Claude pattern.

C. Three-dot menu (⋮)

A ⋮ icon appears on hover. Clicking it opens a small dropdown with "Copy text" and "Flag response". This is the Slack / Discord pattern. Most hidden, but extensible for future actions.

Recommendation: Option A

  • Cleanest UI — no clutter by default
  • Instantly recognizable flag icon
  • Standard pattern for AI chat apps
  • Single click to start report flow

Why not B or C?

  • B: Adds visual weight to every message (thumbs up/down isn't in scope)
  • C: Extra click (open menu → click flag) and hides the action deeper
